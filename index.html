Terminalo komandos

  • ls / dir - išvardina aplankalus ir failus, kurie yra dabartinėje repozitorijoje.
  • cd - changeDirectory - keičia direktoriją, vaikštome tarp aplankalų.
  • TAB - just do it - skirtas užpildyti terminalo komandą arba pateikti pasiūlymus.
  • CTRL C - atšaukia vykdomą veiksmą, nutraukia komandą.
  • vidurinis/dešinys pelės mygtukas / įklijuoja tekstą įkomandinę eilutę.
  • ARROW UP/DOWN - vaikščiojimas per terminalo komandų istoriją

Git komandos

  • git init - inicializuoja git repozitoriją
  • git status - parodomas git repoitorijos statusas
  • git add - pridėti failus prie paruoštų komitinimui failų
  • git commit -m "zinute" - sukuriama kodo versija su žinute
  • git restore - panaikina dar nekomitintus pakeitimus, kurie neaddinti
  • git restore --stage - išima nurodytus failus iš stage zonos (atima pakeitimus, jeigu yra addinti)
  • git push - nusiunčia naujus lokalius commitus į serverį (GitHub)
  • git log - rodo git komitu istorija
  • git checkout commitID - nukeliauja į nurodyta kodo versiją
